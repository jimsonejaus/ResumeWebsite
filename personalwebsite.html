Description

The personal website is a simple website made using HTML and CSS as the final project for the Web Page Design class. It implements the fundamentals and best practices for HTML and CSS. The website is created to be responsive to cater to all device sizes. For the website's layout, we were required to make use of both flexboxes and grids to familiarize ourselves with both of them.

Project Highlights

This project was a very good opportunity for me to showcase what I already know regarding front-end development and a little bit of UI/UX Design. I was able to showcase my personal aesthetic while practicing my knowledge in web development. This project also pushed me to try other front-end projects in my organization due to how much I liked this project.
