Maze Runner

Project Type: Academic Project

My Role: Main Developer

Technologies Used:

Python

Github

Description

Maze Runner is a simple console puzzle game implemented using Python. The goal of the game is to reach the exit point in the maze. Once it is running, it takes in the level of difficulty that the player wants and creates the corresponding grid for the specified level of difficulty. After that, it takes in the values for the starting point and the exit point. It will then display a console drawing of the maze to help you visualize the game. For a specific location in the maze, the directions which the player can go to may be limited and the program will output those directions available to you.

This project was used to test our knowledge and skills in the fundamentals of python programming.

Project Highlights

This was my first major development project. Since it was just the fundamentals of Python, in hindsight, it was not very challenging but I spent quite a lot of time making my code as efficient as possible. The program mainly relies on data structures and string manipulation techniques output the necessary values.
