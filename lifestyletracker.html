Lifestyle Tracker

Project Type: Academic Project

My Role: Main Developer

Technologies Used:

Java

Github

Description

Lifestyle Tracker is a simple Java console and GUI program that monitors the user's calorie intake and physical activities, and creates a report on it. This is a project on object-oriented programming which applies our lessons on the fundamentals of Java. The user can choose the Console or GUI version of the app depending on which file the user runs.

Project Highlights

As this class served as our introduction of object-oriented programming, it presented a new challenge to our programming knowledge. Perhaps the main challenge I faced for this project was my limited knowledge in the syntax for Java as it is a lot more complicated than Python. Though, this taught to be more open to other people's help and feedback as it can really be helpful to see our programs in another perspective. Overall, I found this project to be really challenging and engaging and was definitely necessary to boost my programming knowledge.
