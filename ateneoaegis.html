Description

The Ateneo Aegis website is another website design and development project under the UXSociety Ateneo. It serves as the yearbook portal for graduating students and organizations in the Loyola Schools to schedule their photoshoots and remember the moments that they will cherish for a lifetime. In here, both students and organizations will be able to see the available timeslots where they can schedule their appointments.

Project Highlights

One of the major challenges that we encountered in this project is that we have to create multiple pages from scratch. All the components of the website has to be responsive already and are ready for integration, meaning certain components should be ready to change their states depending on the data found on the back-end. More so, the code that we write also has to be clean and understandable since the team that worked in this project is significantly larger than the other projects I've worked on. Despite all these challenges, as well as the time constraints for the project, our team was still able to finish all our tasks and hand over the project for the integration.
