Description

Blue Mark Delivery Services is our group's final project for our Information management class. Our main task is to incorporate our SQL queries to a simple web page with a database. The SQL queries may be used for inputs, manipulation of data, or fetching information from the database. The main function of our web page is to filter the results shown on the screen by their origin, destination, its service type, package type or any combination of those four. It will then return those filtered results and display them in the website.

This project started with a specification of a company process that was provided to us by our professors. From this company process, we needed to devise a entity relationship diagram that would capture the relationships between the entities that interacts in the process. After that, we transformed the conceptual ER diagram we made into a logical ER diagram with normalized relations so we can use it for our SQL database. Then, the development of the database and the front-end itself started immediately after.

Project Highlights

With this being our official introduction to SQL and information management, it was quite challenging to grasp it at first. However, with enough practice and help also from the internet, we were able to deliver a good product in the end.

Perhaps the main challenge that the developers faced is the integration part between the front-end and the back-end of the web app. Upon selecting the options from the dropdown and clicking the filter button, it should already execute the query for that specific combination of filters. Now, the solution we have come up with is to create a dynamic query which takes in different values depending on the options selected so that we don't have to create a different query for all the possible combinations. This was the more efficient approach to our problem even though it took us a bit longer to implement. This kind of approach will also make it easier for future developers to add, modify, or remove filters when necessary.
